# 🌐 ChatWithWebsite (Powered by Gemini)

ChatWithWebsite is a Streamlit-based web application that allows users to load any static website and ask questions based strictly on its content.

It uses:

- 🔎 Web scraping (Requests + BeautifulSoup)
- ✂️ Text chunking (LangChain Text Splitters)
- 🤖 Google Gemini (gemini-2.5-flash)
- 🎨 Streamlit UI

---

## 🚀 Features

✅ Load any static HTML website  
✅ Ask questions about the website  
✅ Answers generated strictly from scraped content  
✅ Clean chat-style interface  
✅ Gemini Free Tier supported  

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- BeautifulSoup
- Requests
- LangChain Text Splitters
